# Data Analyst Learning Repo

Welcome to the **Data Analyst Learning Repo** 🎉

This repository is designed as a beginner-friendly resource for students preparing for data analyst roles.  
It contains:
- Sample datasets
- SQL practice queries
- Personal learning notes

The purpose of this repo is to maintain a **public learning resource** that anyone can use to practice and improve their data analysis skills.  
More content will be added over time as learning progresses.

---

## Contents
- `sales_dataset.csv` → A small dataset for practice
- `sql_questions.sql` → Beginner SQL queries
- `notes.md` → Learning notes and tips

---

## How to Use
1. Download or clone the repository.
2. Explore the dataset and try running the SQL queries.
3. Read through the notes for guidance and tips.
4. Contribute by adding your own queries or notes!

Happy Learning 🚀
